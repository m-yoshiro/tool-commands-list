# tool-commands-list
よく使うツールのよく使うコマンドをリストアップ

## node.js

### npm
- npm i -y
- npm i -D [module-name]
- npm i -S [module-name]
- npm update -g npm

### nodebrew
- nodebrew ls-remote
- nodebrew install-binary v0.x.x
- nodebrew use v0.x.x
- nodebrew migrate-package v0.xx.xx


## Ruby

### rbenv
- rbenv install -l
- rbenv install 2.x.x
- rbenv global 2.x.x

## Homebrew

### Homebrew
- brew install [name]
- brew tap
- brew tap [name]
